# Realtime Location Sharing

A phase-based MVP for a realtime location sharing mobile computing app. Users will join a room, share foreground location, and see live map markers from other users in the same room.

## Planned Tech Stack

- Expo React Native
- TypeScript
- expo-location
- react-native-maps
- socket.io-client
- Node.js
- Express
- Socket.IO

## Implementation Approach

Implementation is phase-based. Phase 0 prepares the repository structure and documentation only; server and mobile implementation will be added in later phases.
